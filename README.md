# 📊 Sales Intelligence Dashboard (DevOps Pipeline)

Welcome to my professional Data-DevOps project. This repository showcases a fully automated CI/CD pipeline for a Sales Analytics Dashboard.

## 🚀 Live Demo
You can view the live project here: [https://sales-dashboard-devops.onrender.com](https://github.com/ranacse/Sales-Dashboard) *(আপনার Render URL টি এখানে বসিয়ে দিন)*

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3 (Modern Responsive UI)
- **CI/CD Pipeline:** GitHub Actions
- **Code Quality:** SonarCloud (Static Analysis)
- **Deployment:** Render (Cloud Hosting)

## ⚙️ DevOps Workflow
This project follows a modern DevOps methodology:
1. **Commit:** Code is pushed from local machine to GitHub.
2. **Scan:** GitHub Actions triggers a **SonarCloud** scan to ensure code quality and security.
3. **Deploy:** Upon a successful scan, **Render** automatically deploys the latest version to the live URL.

## 📈 Project Overview
This dashboard represents a sales analytics system designed to track product performance, revenue, and categories. It is built to demonstrate how data projects can be integrated into a robust automated pipeline.
